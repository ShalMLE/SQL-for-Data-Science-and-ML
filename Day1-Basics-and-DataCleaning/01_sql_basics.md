# SQL Basics for Data Science & Machine Learning

## 1. Essential Commands
- **SELECT** — extract columns  
- **FROM** — choose table  
- **WHERE** — filter rows  
- **ORDER BY** — sort  
- **LIMIT** — restrict output  
- **DISTINCT** — remove duplicates  

## 2. Comparison Operators
=, !=, >, >=, <, <=, BETWEEN, IN, LIKE

## 3. Logical Operators
AND, OR, NOT

## 4. Aggregations (Preview)
COUNT(), SUM(), AVG(), MIN(), MAX()

## 5. NULL Handling
- COALESCE()  
- IFNULL()  
- NULLIF()

Used for:
- Feature engineering
