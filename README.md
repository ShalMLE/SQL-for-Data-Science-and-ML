This repository showcases a structured, 3-day SQL learning and implementation plan focused on the SQL skills required for **Data Scientist**, **Machine Learning Engineer**, and **Analytics** roles.  
It includes hands-on exercises, dataset cleaning, window functions, business analytics, A/B testing, and ML-ready feature engineering.

The goal of this project is to demonstrate **practical SQL proficiency** through real, production-style workflows used in DS/ML.
